# lecture1 프로젝트 설정

## 역할
- React + MUI 기반 웹 개발 학습 환경
- 단계별 수업 진행

## 문서 참조
@docs/design-system.md
@docs/code-convention.md
@docs/new_project.md

## 개발 환경
- Node.js v20.11.0
- Vite 5.x
- React 18
- MUI v9

## 언어 설정
- 모든 답변은 한국어로 작성
- 코드 설명은 단계별로 친절하게

## 프로젝트 구조
- `docs/` - 디자인 시스템, 코드 컨벤션, 신규 프로젝트 가이드
- `_template_settings/` - MUI 완전 설치된 React 템플릿
