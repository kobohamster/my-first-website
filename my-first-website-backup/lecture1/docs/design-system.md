# 디자인 시스템 (Design System)

## 컬러 팔레트

### Primary Colors
- Primary: `#1976d2` (MUI Blue)
- Secondary: `#dc004e` (MUI Pink)
- Background: `#ffffff`
- Surface: `#f5f5f5`

### Text Colors
- Primary Text: `rgba(0, 0, 0, 0.87)`
- Secondary Text: `rgba(0, 0, 0, 0.6)`
- Disabled: `rgba(0, 0, 0, 0.38)`

## 타이포그래피

- Font Family: Roboto, Helvetica, Arial, sans-serif
- h1: 2.125rem / 500
- h2: 1.5rem / 400
- body1: 1rem / 400
- caption: 0.75rem / 400

## 간격 (Spacing)
- Base unit: 8px
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px

## 컴포넌트 가이드라인

### Buttons
- Contained: 주요 액션
- Outlined: 보조 액션
- Text: 최소한의 강조

### Cards
- elevation: 1 (기본), 3 (호버)
- border-radius: 4px
